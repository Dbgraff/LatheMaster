# LatheMaster

- Учебный проект, релализованный в рамках производственной практики. Должен помочь школьникам 10-11 классов узнать из чего состоит токарный станок и посмотреть как он работает.

Сейчас в проекте реализованы: 
- Теория по ТБ,
- Описание главных частей станка
- Описание производимых видов работ
- Симуляция работы станка в 3д
